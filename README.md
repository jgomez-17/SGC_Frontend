# Sistema de Gestión de Créditos (FRONTEND)

Este proyecto es una aplicación web para gestionar créditos de clientes, incluyendo creación, visualización, filtrado y ordenamiento.  

Desarrollada con Nextjs, Expressjs y postgresql
---

### paso 1: Clonar el repositorio  

-- git clone <URL_DEL_REPOSITORIO>

### paso 2: abrir el proyecto

-- cd <nombre-del-proyecto>

### paso 3: instalar las dependencias

-- npm install

### paso 4: ejecutar el proyecto

-- npm run dev 

y verlo en http://localhost:3000/

###### LUEGO CREAR EL ARCHIVO (.env.local) EN LA RAIZ DEL PROYECTO #### y pegar las sgtes variables

NEXT_PUBLIC_API_URL=http://localhost:4000/api/creditos
